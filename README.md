# LetsWork
	
## Authors
Charles Guertin and Ludovic Lachance

## Dependencies

Composer must be installed on your machine.

To install composer dependencies:
```
composer install
```

## How to run the application

### With Vagrant

```
vagrant up
```

### With Docker

```
docker-compose up
```

## Technologies

- Laravel
- MySQL
- Laravel Blade
- Vue.js
